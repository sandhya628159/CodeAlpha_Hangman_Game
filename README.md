# 🎮 CodeAlpha Hangman Game

A classic, interactive word-guessing game built using Python. The program challenges the user to guess a hidden word, letter by letter, within a limited number of attempts.

---

## ✨ Features
* 🔠 **Dynamic Word Selection:** Randomly picks words for a fresh gameplay experience every time.
* 📉 **Attempt Tracker:** Displays visual feedback or remaining attempts after each guess.
* 💻 **Interactive CLI:** Simple and clean command-line interface for smooth playing.

---

## 🛠️ Tech Stack
* **Language:** Python 3.x

---

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```

2. **Navigate to the directory:**
   ```bash
   cd CodeAlpha_Hangman_Game
   ```

3. **Run the game:**
   ```bash
   python hangman.py
   ```
   *(Note: Change `hangman.py` to your actual filename if it is different)*
